# 奶粉蛋白质计算器 (Milk Protein Tracker)

这是一个为家长设计的专业级奶粉蛋白质摄入追踪工具。它可以根据宝宝体重自动计算每日蛋白质需求目标，并支持多种奶粉规格的精确记录，帮助科学喂养。

## 🌟 核心功能

- **智能需求计算**：基于体重自动生成每日蛋白质摄入目标。
- **多规格支持**：预设常用奶粉规格（如华夏2号、纽贝福、华夏蛋白粉），并支持自定义添加新规格。
- **高精度输入**：支持 0.2 粒度的加减调整，同时支持手动精确输入克数/勺数。
- **视觉化追踪**：环形进度条和底部进度条实时展示今日完成百分比。
- **数据持久化**：所有自定义规格和记录均保存在本地浏览器（localStorage），无需注册登录。
- **极致体验**：大字体设计，适配移动端单手操作。

## 🧪 计算公式

应用遵循以下专业公式：
- **目标蛋白质 (g)** = 体重 (kg) × 2.1 × 0.8
- *注：输入单位为“斤”，系统自动转换为“kg”进行计算（1kg = 2斤）。*

## 🛠️ 技术栈

- **前端框架**：React 19 (ESM)
- **样式处理**：Tailwind CSS (CDN)
- **图表库**：Recharts
- **语言**：TypeScript
- **构建/部署**：基于纯浏览器 ES 模块，无需复杂编译环境。

## 🚀 部署与运行

### 方式一：直接运行 (推荐)
由于项目使用了 `importmap` 和原生 ES 模块，您只需在包含 `index.html` 的目录下启动任何静态 Web 服务器即可：

```bash
# 使用 Python 快速启动
python -m http.server 8000
```
然后在浏览器访问 `http://localhost:8000`。

### 方式二：静态托管
将项目根目录下的所有文件上传至 GitHub Pages、Vercel 或 Netlify 等静态托管平台即可直接在线使用。

## 📝 使用指南

1. **输入体重**：在页面顶部输入宝宝当前的体重（单位：斤）。
2. **记录摄入**：点击奶粉规格右侧的 `+` 或 `-` 调整摄入量，或直接在中间数字框输入。
3. **查看进度**：蓝色卡片展示目标与现状，底部固定栏提供实时百分比。
4. **管理规格**：点击“添加规格”可录入新的奶粉数据（蛋白质含量需按每100g含多少g计算）。
